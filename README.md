## Introduction

### Installation

```/bin/sh
bin/docker init
```

### Development

To develop, run the service:

```/bin/sh
bin/docker start
```

When finising stop the service:

```/bin/sh
bin/docker stop
```

When building assets:

```/bin/sh
bin/gatsby build
```
